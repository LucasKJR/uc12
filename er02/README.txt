UC12 - Projeto de Backend para cadastro de Pessoa Física e Jurídica com gravação em formato CSV

Descrição:
Este projeto tem como objetivo a criação de um sistema de cadastro de Pessoa Física e Jurídica. Ao final do processo, os dados serão salvos em formato CSV.

Tecnologias:
Serão utilizados DOTNET e C#, seguindo o padrão de camelcase para facilitar a leitura do código.

Organização do Projeto:
Serão criadas as classes Endereço, Pessoa, Pessoa Física e Pessoa Jurídica. A classe Endereço será utilizada dentro da classe Pessoa. Serão criadas interfaces como IPessoa, IPessoaFisica e IPessoaJuridica. O terminal será estilizado para uma melhor visualização das opções de cadastro de PF e PJ e para a listagem de PF e PJ.

Execução da Aplicação:
Serão criados métodos para cadastrar e listar Pessoa Física, com nome, CPF e endereço. Também serão criados métodos para cadastrar e listar Pessoa Jurídica, com razão social, CNPJ e endereço. A execução do terminal de cadastramento será feita através do terminal do VSCode com o comando dotnet run.

Pré-requisitos de instalação:
Para o desenvolvimento deste projeto, foi utilizado o VSCode com a extensão C# v1.25.
